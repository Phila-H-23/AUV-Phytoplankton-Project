This folder contains all the Fusion files needed to make 3D print the actuation system. Below is a descripion of each file and what is in it:

Assembly.f3z - Final assembly of all individual parts to see what the final integration should look like. Not for printing purposes.
Gears.f3z - CAD file of the 2 spur gears at the motor input
Guide_Stand.f3z - CAD file of GuideStand. Thrust and Radial Bearing stand must be glued on manually once you have achieved the desired gearbox positioning 
Lead Screw.f3z - 
Lead Screw Head.f3z - CAD file of disc that needs to be glue to the top of the Lead Screw to Interface with the Piston Head properly
Motor.f3z - CAD model of the motor. Won't be printed but there for reference.
Piston head.f3z - CAD file of main piston head with groove to interface with the Screw Head 
Radial Bearing Stand.f3z - Stand to place bearing on. Use 607 bearing so that it fits
Thrust Bearing Stand.f3z - Stand to place thrust bearing in. Use 51100 thrust bearing so that it fits.
Worm Gear and Drive.f3z - CAD file of the Worm wheel and Nut with threads added to the inside of the worm wheel.

Modifications that might be required:
-Offset used for the threading in the "Worm Gear and Drive.f3z" may need to be adjusted according to your printer and printer accuracy. The more accurate the printer, the less offset is required.
-Most parts may need some sanding to get them to fit as desired, especially the bearing stands.
-Motor Bracket and and groove for the gears in the guidestand might need to be adjusted according to the motor used.
